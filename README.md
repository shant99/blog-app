## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Additional Commands

Prisma Studio run

```
npx prisma studio
```

Prisma types generate

```
npx prisma generate
```

Prisma model db add

```
npx prisma db push
```
